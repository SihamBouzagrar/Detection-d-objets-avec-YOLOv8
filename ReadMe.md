# 🎥 Détection d'objets avec YOLOv8

Ce projet montre comment utiliser **YOLOv8** (Ultralytics) pour appliquer la détection d’objets sur des **images** et des **vidéos**, en utilisant **Python**, **OpenCV**, et **Pandas** dans un environnement comme Google Colab.

---

## 🚀 Installation

Avant de lancer le code, installe les dépendances nécessaires :

```bash
pip install ultralytics tensorflow keras opencv-python pandas

Structure du projet

    notebook.ipynb : script principal (Colab ou Jupyter).

    video.mp4 : exemple de vidéo d’entrée.

    output.mp4 : vidéo annotée en sortie.

    README.md : documentation du projet.

🛠 Bibliothèques utilisées

    Ultralytics : fournit les modèles YOLOv8 pré-entraînés pour la détection d’objets.

    TensorFlow / Keras : frameworks de deep learning (utilisés pour extensions ou intégrations).

    OpenCV : lecture/écriture de vidéos et manipulation d’images.

    Pandas : organisation et analyse des résultats de détection.



✅ Objectifs pédagogiques

    Comprendre l’utilisation de YOLOv8 pour la détection d’objets.

    Appliquer la détection sur des images et vidéos.

    Extraire et analyser les résultats avec Pandas.

    Visualiser les résultats dans Colab.
